# 🏥 AI-Based Electronic Health Record (EHR) System

## 🚀 Overview
This project is a full-stack AI-powered Electronic Health Record system that manages patient data and provides disease prediction using Machine Learning models.

## 🧠 Features
- Patient & Doctor Management
- Medical Records Storage
- Authentication & Authorization
- AI Predictions:
  - Diabetes Prediction
  - Heart Disease Prediction
  - ECG Abnormality Detection

## 🏗️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js (Express)
- ML Service: FastAPI (Python)
- Database: MongoDB
